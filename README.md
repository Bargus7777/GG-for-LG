Установка приложения GG for LG

Скачать архив приложения:

https://github.com/Bargus7777/GG-for-LG/archive/refs/tags/v2.0.9.zip

Алгоритм установки :
Установить LG webOS Dev Manager на компьютер.
Установить и включить Developer Mode на телевизоре.
Установить приложение "GG for LG" через LG webOS Dev Manager на телевизор.

Шаг 1. Установка LG webOS Dev Manager

Скачать установщик:

https://github.com/webosbrew/dev-manager-desktop/releases/download/v1.13.3/webOS.Dev.Manager_1.13.3_x64_en-US.msi

Установите программу и запустите её.

Шаг 2. Установка и включение Developer Mode

Установка по инструкции:

https://webos-forums.ru/post182943.html?ysclid=mnxbtuey9w145142664

В приложении Developer Mode включите:

Developer Mode — ON
Key Server — ON

Перезапустите телевизор.

Шаг 3. Установка приложения "GG for LG"

Установите приложение "GG for LG" с компьютера через LG webOS Dev Manager.

Для установки используйте файл:

GG-for-LG-X.X.X.ipk
